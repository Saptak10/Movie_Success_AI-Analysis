Overview

Data‑Driven Movie Success Analysis

From heterogeneous web data to explainable models that classify movies as flops, average performers, or hits.

Studios and platforms routinely invest tens of millions into projects whose commercial outcome is uncertain. This project brings together budgets, revenues, release timing, genres, cast and crew credibility, and pre‑release attention signals to understand why some titles break out while others underperform.

The dataset is built from public sources (IMDb, TMDb, The Numbers, Wikipedia, Wikimedia pageviews, and IMDb bulk datasets), cleaned and normalised into an analysis‑ready table that links each film to rich economic and creative metadata.

What This Project Delivers
  • Exploratory insights into how budget, genre, release timing, and audience interest relate to box‑office outcomes.
  • Classification models that label each movie as Flop, Average, or Hit using tree‑based algorithms.
  • Feature importance analysis to explain which creative and commercial factors drive predictions.
  • Interactive front‑end where users can explore past films and run “what‑if” scenarios for new releases.

Methodology

From raw web data to movie success predictions

1
Data Collection

Gather budgets, box‑office figures, rich metadata, cast and crew information, and pre‑release activity from IMDb, TMDb, The Numbers, Wikipedia, and Wikimedia pageviews.

2
Wrangling & EDA

Clean and normalize sources, resolve duplicates, align identifiers, and explore distributions, correlations, and segment patterns to uncover data issues and signal.

3
Feature Engineering

Build economic and creative features such as log‑scaled profitability, temporal indicators, genre encodings, and actor/director credibility scores from historical filmography.

4
Modeling

Cluster movies with similar profiles and train classifiers to predict Flop, Average, or Hit while inspecting accuracy, class‑wise metrics, and feature importance.

5
Deployment & Frontend

Expose cleaned data and models through a web interface where users can explore past films, inspect predictions, and run “what‑if” scenarios for new projects.
